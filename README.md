# LedgerLive

**LedgerLive** is a lightweight inventory & sales management system for small businesses.  
Track products, suppliers, purchases, and sales; auto-update inventory; view dashboards; and export reports.

- **Interactive dashboards & reports** (daily/weekly/monthly)
- **Export** reports to CSV/PDF
